
En esta carpeta se suministran los dos instaladores de la aplicacion nativa (app cliente y app server). El contenido de cada carpeta del instalador tiene una carpeta OUTPUT donde esta el instalador creado por el programa INNO SETUP, otra carpeta con el nombre de la aplicacion que es la que emplea en INNO SETUP para generar el instalador. Alli se encuentran las librerias de python requeridas para el autoejecutable de la clase principal, las dll y otros archivos necesarios. El manifest que viajara con el instalador, este archivo json es necesario para la invocacion de la aplicacion ya instalada y permitir la conexión con la extension. Por ultimo, un archivo de generación del INNO SETUP que se empleo para generar el instalador.
Una vez instalada la app, el contenido que se vera en la carpeta instalada, es tal como se ve el contenido en la carpeta donde esta el autoejecutable.
Cada autoejecutable fue creado empleando "PYINSTALLER". Esta herramienta puede originar falsos positivos de virus una vez creado el autoejecutable.
